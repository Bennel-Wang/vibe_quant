# 股票综合评分体系 v2.0

## 一、概述

综合评分由**价值体系（Value System）**和**趋势体系（Trend System）**两套独立评分构成，最终取加权融合值。两套体系各自独立给出 0–100 分，最终综合得分公式为：

```
total_score = 0.7 × max(value_score, trend_score) + 0.3 × min(value_score, trend_score)
```

- 理论最低：两套均为 0 → 0 分
- 理论最高：两套均为 100 → 100 分
- 主导类型判定：差值 > 5 分则判定为某一体系主导，否则均衡

**API 端点：** `GET /api/stock/<code>/factors`  
**数据要求：** 至少 60 个交易日；PE/PB 仅 A 股个股可用（其他市场类型默认中性）

## 三、价值体系（Value System）

**核心哲学：** 寻找估值极低、超跌严重、情绪绝望、逻辑打透、但近期开始止跌企稳的标的。

### 权重分配

| 维度 | 权重 | 核心问题 |
|------|------|----------|
| V1：估值极底 | 15% | 估值历史上没法更低了吗？ |
| V2：多周期超卖 | 22% | 日/周/月RSI都在低位吗？ |
| V3：历史价格底部 | 15% | 股价在500日历史低位吗？ |
| V4：深度调整超跌 | 12% | 中期跌幅已经很惨了吗？ |
| V5：筑底信号 | 15% | KDJ/WR/布林/量能出现底部信号？ |
| V6：空头钝化 | 13% | 近期止跌，下跌节奏放缓？ |
| V7：形态安全边际 | 8% | 低点是否在抬升（不是一浪比一浪低）？ |

## 五、综合评分公式

```
value_score  = 0.15×V1 + 0.22×V2 + 0.15×V3 + 0.12×V4 + 0.15×V5 + 0.13×V6 + 0.08×V7
trend_score  = 0.12×T1 + 0.22×T2 + 0.18×T3 + 0.12×T4 + 0.13×T5 + 0.13×T6 + 0.10×T7
total_score  = 0.7 × max(value_score, trend_score) + 0.3 × min(value_score, trend_score)
```

**主导类型：**
- `trend_score > value_score + 5` → 趋势主导
- `value_score > trend_score + 5` → 价值主导
- 否则 → 趋势+价值均衡

## 六、边界处理

- 所有子分 `clamp(v, 0, 100)`
- PE/PB 不可用：V1=50，T1=60
- PE<0（亏损）：V1特殊处理=10，T1=15
- 数据不足60行：涉及历史比较因子默认50
- 500日分位有效数据<10条时默认50
