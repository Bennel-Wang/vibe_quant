# 大盘阶段判断说明

> 文件路径: `quant_system/market_regime.py`  
> 基准指数: 上证综指 (`000001.SH`)  
> 判断结果: **乐观 / 混沌 / 悲观 / 极度悲观**

---

## 一、判断流程概览

```
获取上证综指近400天日线数据
        ↓
计算5项量化指标并分别打分
        ↓
汇总综合评分 (范围 -100 ~ +100)
        ↓
按阈值划分4个大盘阶段
        ↓
输出: 阶段标签 + 评分 + 各项细节
```

---

## 二、5项指标与评分权重

| # | 指标 | 权重 | 得分范围 | 说明 |
|---|------|------|----------|------|
| 1 | **20日涨幅** (`idx_ret_20`) | 30% | −30 ~ +30 | 近一个月趋势，最重要 |
| 2 | **60日涨幅** (`idx_ret_60`) | 20% | −20 ~ +20 | 近一个季度趋势 |
| 3 | **均线排列** (4层MA) | 25% | −25 ~ +25 | 价格位置与多/空排列 |
| 4 | **RSI(14)** | 15% | −15 ~ +15 | 超买/超卖状态 |
| 5 | **量比** (`volume_ratio`) | 10% | −10 ~ +10 | 成交量活跃度 |

合计权重：**100%**，综合评分区间：**−100 ~ +100**

---

## 三、各项指标计算细节

### 3.1 涨幅评分（指标1、2）

```python
idx_ret_20 = (当前收盘 - 20天前收盘) / 20天前收盘 × 100   # 单位：%

score_20d = clip(idx_ret_20 × 5,  -30, +30)   # 权重30%
score_60d = clip(idx_ret_60 × 2,  -20, +20)   # 权重20%
```

**举例**：上证综指20日涨幅 +5% → 得分 = 5×5 = **+25分**  
**举例**：20日涨幅 −7% → 得分 = −7×5 = −35，截断为 **−30分**

---

### 3.2 均线排列评分（指标3）

对4个条件逐项加分，满分25分，最后中心化：

| 条件 | 分值 |
|------|------|
| 当前价 > MA5 | +5 |
| MA5 > MA20   | +7 |
| MA20 > MA60  | +8 |
| MA60 > MA120 | +5 |
| **最大原始分** | **25分** |

```python
ma_score_raw ∈ [0, 25]
ma_score = (ma_score_raw - 12.5) × 2   # 中心化后放大，范围 -25 ~ +25
```

**均线趋势描述**（展示用，不影响评分）：
- `多头排列`：价格 > MA5 > MA20 > MA60  
- `空头排列`：价格 < MA5 < MA20 < MA60  
- `均线纠缠`：其他情况

---

### 3.3 RSI(14) 评分（指标4）

```python
rsi_score = clip((RSI14 - 50) × 0.3,  -15, +15)
```

| RSI值 | 得分举例 | 状态描述 |
|-------|----------|----------|
| 80 (超买) | (80-50)×0.3 = +9 | 超买 |
| 50 (中性) | 0 | 中性 |
| 25 (超卖) | (25-50)×0.3 = −7.5 | 超卖 |

> 注意：RSI超买反而得分高，因为在本系统语义中，超买=市场强势=偏乐观。

---

### 3.4 量比评分（指标5）

```python
vol_score = clip((volume_ratio - 1.0) × 10,  -10, +10)
```

| 量比 | 得分 | 含义 |
|------|------|------|
| 2.0 | +10 | 成交活跃，市场情绪高涨 |
| 1.0 | 0   | 正常水平 |
| 0.5 | −5  | 缩量，市场观望 |

---

## 四、综合评分与阶段划分

```
综合评分 = score_20d + score_60d + ma_score + rsi_score + vol_score
范围: -100 ~ +100
```

| 阶段 | 评分区间 | 描述 | 操作策略倾向 |
|------|----------|------|-------------|
| 🟢 **乐观** | > +30 | 大盘趋势向上，动量强 | 短线/成长股动量策略 |
| 🟡 **混沌** | −10 ~ +30 | 方向不明，震荡整理 | 中线/震荡策略 |
| 🔴 **悲观** | −40 ~ −10 | 趋势向下，弱势格局 | 防御股持仓，非防御空仓 |
| ⚫ **极度悲观** | ≤ −40 | 市场极弱，大跌模式 | 长线价值布局/超级长线 |

---

## 五、各阶段适配的策略

| 大盘阶段 | 适用股票类型 | 推荐策略 |
|----------|-------------|----------|
| 乐观 | 成长股、高弹性股 | MACD金叉、成长股动量、短线突破 |
| 混沌 | 周期股、价值股 | 中线2、震荡RSI、布林带均值回归 |
| 悲观 | 防御股（公用事业、消费） | 防御超跌买入；其余品种→**空仓** |
| 极度悲观 | 价值股（PE极低） | 长线4421策略；其余→**空仓** |
| 任意（超级长线）| 高质量蓝筹 | 超级长线持有（无视大盘） |

## 六、数据来源与更新频率

- **数据源**：本地历史日线CSV（每日收盘后由调度器 `data_update` 任务更新）
- **基准指数**：上证综指 `000001.SH`
- **更新触发**：
  - 页面访问时实时计算（调用 `/api/market/regime`）
  - 每日 15:50 定时任务 `大盘阶段判断及对应策略推荐` 自动运行并发送微信通知

---

## 七、局限性说明

1. **不含节假日过滤**：交易日判断用简化版（周一至五），不排除法定节假日
2. **量比精度**：`volume_ratio` 依赖近20日均量，若历史数据不足则偏差较大
3. **RSI方向性**：本系统RSI评分与常见"高RSI看空"不同，高RSI反映市场强势，被设计为正贡献
4. **单一指数**：仅用上证综指，不能完全反映深市/创业板/港股情况
5. **滞后性**：所有指标基于历史收盘价，不含盘中实时预测能力
