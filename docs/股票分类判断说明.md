# 股票分类判断说明

> 文件路径: `quant_system/stock_classifier.py`  
> 分类结果: **成长型 / 防御型 / 周期价值型**

---

## 一、分类流程

```
获取股票信息（行业、代码、名称）
        ↓
方法1：行业映射（优先级最高）
→ 行业名称存在且在映射表中？
        ↓ YES          ↓ NO
   直接返回         计算20日波动率 + Beta
                        ↓
              方法2：波动率 + Beta判断
              → 数据充足（≥20条历史数据）？
                    ↓ YES          ↓ NO
              阈值规则判断       方法3：默认
                                → 周期价值型
```

---

## 二、方法1 — 行业映射（最高优先级）

行业名称直接与下表对照，有匹配即使用，**不再计算波动率**。

### 成长型（🚀 Growth）
| 行业 |
|------|
| 半导体、电气设备、通信设备、元器件 |
| 软件服务、互联网、IT设备、计算机应用 |
| 电子元件、光学光电子 |
| 汽车整车、汽车配件 |
| 医疗器械、化学制药、生物制品 |
| 专用机械、航天航空、仪器仪表 |

**逻辑**：成长股通常来自技术创新、消费升级、医疗科技等赛道，盈利增速高、估值弹性大。

---

### 防御型（🛡️ Defensive）
| 行业 |
|------|
| 水力发电、火力发电、电力、燃气、供水供气 |
| 银行、保险 |
| 白酒、食品饮料、乳品、调味品 |
| 中药、医药商业 |
| 环保工程 |
| 高速公路、铁路运输、机场、港口 |
| 零售、商业百货 |

**逻辑**：防御股在经济下行时仍有稳定现金流，Beta低，适合悲观/极度悲观大盘阶段持仓。

---

### 周期价值型（🔄 Cyclical/Value）
| 行业 |
|------|
| 化工原料、基础化学 |
| 钢铁、煤炭开采、石油开采 |
| 有色金属 |
| 水泥、建筑材料 |
| 全国地产、区域地产 |
| 工程机械、船舶制造 |
| 纺织、家居用品、造纸 |
| 农林牧渔 |
| 证券、多元金融 |

**逻辑**：与宏观经济、大宗商品价格强相关，盈利周期性明显，适合混沌/震荡大盘阶段。

## 三、方法2 — 波动率 + Beta（行业无法匹配时使用）

### 数据计算

```python
# 取最近 120 天日线数据（至少需要 20 条）
vol_20d = 近20日日收益率的标准差        # 20日波动率
beta    = cov(股票日收益, 上证日收益) / var(上证日收益)   # 相对上证综指
```

### 分类规则

| 条件 | 分类 | 含义 |
|------|------|------|
| `vol_20d > 0.03` **且** `beta > 1.2` | 🚀 成长型 | 高弹性、跑赢大盘 |
| `vol_20d < 0.02` **且** `beta < 0.8` | 🛡️ 防御型 | 低波动、跑输大盘 |
| 其他情况 | 🔄 周期价值型 | 中等波动，走势与大盘接近 |

#### 参数说明

| 参数 | 说明 | 典型值参考 |
|------|------|-----------|
| `vol_20d` | 20日日收益标准差 | 防御股 ~0.010～0.018；成长股 >0.030 |
| `beta` | 与上证综指的协动系数 | 防御股 ~0.5～0.8；成长股 ~1.2～2.0 |

---

## 四、方法3 — 默认（数据不足时）

当历史数据不足20条时，无法计算波动率和Beta，默认归为**周期价值型**，等待数据累积后重新判断。

---

## 五、分类与大盘阶段的对应关系

| 大盘阶段 | 适合操作的股票类型 | 理由 |
|----------|------------------|------|
| 🟢 乐观 | 成长型为主 | 牛市中高Beta股票涨幅更大 |
| 🟡 混沌 | 周期价值型、成长型均可 | 无明显方向，需分散 |
| 🔴 悲观 | 防御型（其余空仓） | 熊市中低Beta股票抗跌 |
| ⚫ 极度悲观 | 防御型持仓，或全空仓等待长线布局 | 保存实力，等待底部 |
| 任意（超级长线）| 成长型、防御型均可 | 长期持有，无视短期波动 |

---

## 六、局限性说明

1. **行业名称依赖数据源**：行业字段来自股票基本信息，不同数据源的行业分类名称可能有差异（如"软件服务"vs"计算机软件"），若不完全匹配则跳过方法1
2. **Beta基于上证综指**：深证/港股成分股的Beta相对上证可能失真，港股高成长股可能被误判为周期型
3. **短期波动率波动大**：在大幅波动的市场（如熔断期），vol_20d会虚高，成长股可能被误判为防御型
4. **历史数据要求**：至少需要20条日线数据才能使用方法2，新上市股票默认为周期价值型
5. **不考虑财务指标**：当前版本不使用PE/PB/ROE等财务数据，仅基于价格行为和行业归属
